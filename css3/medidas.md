### Max/Min width
```css
main{
    width:100vw;
    height: 100vh;
    background-color: purple;
}

section{
    width: 80%;
    min-width: 320px;
    max-width: 500;
    min-height: 500px;  /* permite crecer la altura */
    margin: 0 auto;
    background-color: red;
}
```
